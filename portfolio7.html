TLDR: Custom Zombies map for Call of Duty: Black Ops III

Horde survival level made using the Radiant Editor using gameplay and assets from Call of Duty: Black Ops III. Emphasis on flow and circulation.

Responsibilities:
  • Design level around preexisting game mechanics.
  • Learn how to use the Radiant map editor.
  • Extrapolate feedback from 4-player level playtests.
Links: GitHub







Persistence of Memory

Game Introduction:

The level is built to be played from many different perspectives, players stats are very dynamic and I want those stats to take them to different areas of the map.

Design Philosophy:

I designed the level to have strong verticality, along with lots of spots where you can either camp or train enemies, but with more valuable spots having riskier exits.

Level Introduction:

This level is themed after "The Persistence of Memory" by Salvador Dali, with melting clocks and a color scheme meant to be remeniscent of it. This game is still in progress so this website will be updated over time.
